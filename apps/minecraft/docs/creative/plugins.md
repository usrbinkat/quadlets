# Creative World Plugins

Open sandbox for building. Peaceful, flight, unlimited resources. The art
studio — comfort and ergonomics over stakes.

**Persona:** Players who want to build without pressure. Command blocks
enabled, no grief concern (open canvas), power tools for large-scale
construction. Structures built here can be voted for copy/paste into
survival via WorldEdit/FAWE.

## Not Included (vs survival)

- GriefPrevention — open canvas, no land claims
- Orebfuscator/anti-xray — no resource scarcity
- mcMMO — no progression/combat
- ToolStats — no attachment to gear in creative
- HarvestMaster — instant creative inventory

## Player Requirements

None. All plugins are server-side only. Join with vanilla Minecraft 26.1.2.

## Governance Note

WorldEdit/FAWE are permission-gated via LuckPerms. Trusted builders get
`worldedit.*` permissions. Structures can be exported as schematics for
community vote to import into the survival world.
