# Modded World — NeoForge Mods

Enhanced survival with modern amenities. NeoForge 1.21.8. Hard difficulty,
state-of-the-art survival experience.

**Persona:** Players who want deeper progression without losing the survival
feel. Tech, exploration, and performance mods create a best-of-breed
survival experience beyond what vanilla offers.

## Player Requirements

Players must install NeoForge 1.21.8 on their client. The performance and
infrastructure mods above are server-side only — the client needs only a
matching NeoForge version with no additional mods for the phase 1 server-side
stack.

Terralith is a datapack that affects worldgen server-side. The client
renders the new biomes using vanilla block textures — no client mod needed.
